# Activate Themes

Ketika tema atau template Anda sudah terpasang di CBMS, maka langkah selanjutnya adalah mengaktifkannya

---

- [Mengaktifkan Client Area Template](#activateClient)
- [Mengaktifkan Order Form Template](#activateOrder)

<a name="activateClient"></a>

## Mengaktifkan Client Area Template

> 1. Masuk ke CBMS Admin Area.
> 2. Arahkan ke menu pada sidebar **Setup > General Settings**.
> 3. Pada Tab **General** terdapat pilihan **Template** lalu pilih dengan nama tema yang sudah dibuat.
> 4. Klik **Save Changes** untuk melihat perubahan yang ada pada client area.

---

<a name="activateOrder"></a>

## Mengaktifkan Order Form Template

Ada dua tempat di Admin Area CBMS di mana Anda dapat mengatur Order Form Template:

> - Anda bisa memilih secara *default* Order Form Template di **Setup > General Settings** pada Tab *Ordering*.
> - Anda juga bisa memilih Order Form Template secara spesifik hanya untuk product group tertentu di **Setup > Product Config > Product/Services**.
